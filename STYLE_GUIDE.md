# Audiophile Style Guide

This document outlines the design system for the Audiophile project, including colors, typography, buttons, input fields, and radio selections.

## Colors

- **Primary**: #D87D4A
- **Primary Hover**: #FBAF85
- **Secondary**: #101010
- **Light**: #F1F1F1
- **Light Lighter**: #FAFAFA
- **White**: #FFFFFF
- **Black**: #000000
- **Error**: #CD2C2C
- **Border**: #CFCFCF

## Typography

All headings are in uppercase with Manrope Bold font.

- **H1**: 56px, Line Height: 58px, Letter Spacing: 2px
- **H2**: 40px, Line Height: 44px, Letter Spacing: 1.5px
- **H3**: 32px, Line Height: 36px, Letter Spacing: 1.15px
- **H4**: 28px, Line Height: 38px, Letter Spacing: 2px
- **H5**: 24px, Line Height: 33px, Letter Spacing: 1.7px
- **H6**: 18px, Line Height: 24px, Letter Spacing: 1.3px
- **Sub Title**: 13px, Line Height: 25px, Letter Spacing: 1px
- **Overline**: 14px, Line Height: 19px, Letter Spacing: 10px (Regular weight)
- **Body**: 15px, Line Height: 25px (Medium weight)
